# सुश्रोता · Su-shrotā

**Speech-to-text engine for traditional Sanskrit.**

Prof. Prathosh A P · Indian Institute of Science, Bengaluru · prathosh@iisc.ac.in

A research demo: a fine-tuned IndicConformer acoustic model transcribes spoken
Sanskrit (recitation / śāstra) to Devanagari, with an optional corpus-backed
suggestion layer for one-tap corrections.

## Try it
Open the hosted page (GitHub Pages) on a phone or desktop with a microphone, grant
mic access, tap the mic, and recite. The transcript appears live; tap ⏸ to close a
line with a daṇḍa, ◼ to finish. Underlined words offer suggestions (toggle in ⚙︎);
tap anywhere to edit. **Copy** / **Share** to export.

## Notes
- This is an early demo for quality evaluation — expect errors, especially on rare
  compounds and fast chant.
- Requires network (the acoustic model runs on a server; the page only records audio).
- With consent, recordings + corrections may be stored to improve the model.
